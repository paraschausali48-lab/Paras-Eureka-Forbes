# Eureka Forbes Direct Sales Website Starter

This workspace contains a simple website template you can open and edit in VS Code.

## How to use

1. Open this folder in Visual Studio Code.
2. Install the `Live Server` extension if you do not already have it.
3. Open `index.html`.
4. Right-click the editor and choose `Open with Live Server`.
5. Edit product names, prices, and contact details directly in `index.html`.

## What to customize next

- Replace the example SKUs with your actual selected products.
- Add more cards under the `products` section.
- Update contact details in the `contact` section.
- Change the hero image by replacing the image URL in `index.html`.

## Tips for a better site

- Use clean product photos and clear feature bullet points.
- Keep the page simple so customers can quickly see the SKU and how to contact you.
- Reuse the same brand colors across the page for a polished look.
